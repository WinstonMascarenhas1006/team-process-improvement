# Team working agreement (draft)

Use this as a starting point in the first planning session. Edit with the team, do not paste it in unchanged.

## Meetings

- Daily standup: 15 minutes, same time, camera optional but be present
- Planning: max 90 minutes for a two week sprint
- Retro: 45 minutes, one concrete action item written down before we leave

## Work in progress

- No more than two stories in progress per person
- If a task is blocked for a day, it goes on the impediment log

## Quality

- Definition of Done includes: code reviewed, tests pass locally, short note in the PR
- Merges to main only when CI is green

## Communication

- Questions about scope go to the product owner, not random side chats
- Decisions that affect everyone get a one line summary in the team channel after the meeting
